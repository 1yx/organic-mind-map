# 博赞思维导图产品指导原则

本文件只保留产品必须遵守的规则。UI、交互、算法、导入导出都不得违背这些原则。

---

## 一、核心法则

1. **横向纸张**
   默认使用 A3/A4 横向画布，并保留明确边界感。

2. **中心图像**
   每张图都必须从中心图像开始，不能只有中心文字。中心图像至少 3 色。
   Phase 1 例外：为优先验证“图像化中心 + 有机布局”主链路，来自受控矢量库或 AI 选择的中心 SVG 可以是单色；但中心仍必须是图像或明确视觉符号，不能退化为普通纯文字中心。3 色要求作为后续严格合格标准恢复。

3. **图像优先**
   整张图应广泛使用图像、符号、视觉标记，不能退化为纯文字提纲。

4. **关键词大写**
   英文关键词默认使用大写。

5. **一词一分支**
   每条分支只承载一个关键词，不写短语，不写句子。

6. **曲线渐细**
   分支必须自然弯曲，并从根部到末梢逐渐变细。

7. **线字等长**
   分支物理长度应与其上的词或图像长度匹配。

8. **多色编码**
   颜色不是装饰，而是分类和记忆锚点。主分支之间必须有清晰颜色区分。

9. **非线性关联**
   允许通过箭头和连接线表达跨分支关系。

10. **留白清晰**
    布局必须优先保证可读性、节奏和留白，不能为了塞内容而牺牲清晰度。

---

## 二、进阶法则

1. **主图与子图**
   复杂主题应拆为多个子图，并由主思维导图承担索引作用。

2. **BOIs 与顺序**
   支持基本排序理念，也支持在必要时给分支编号。

3. **云朵边界**
   完整分支可用云状边界包裹，形成组块记忆。

4. **空白线条**
   允许插入空白分支，作为继续联想的占位符。

5. **引导式发散**
   可使用“为什么、是什么、哪里、谁、如何”等提示词帮助继续展开。

6. **创意保护**
   不压制跳跃、反常或看似荒诞的联想。

7. **个人风格**
   允许个性化色彩、图像和符号风格，但不得破坏核心法则。

8. **复习与重绘**
   产品应支持周期复习，以及基于记忆的快速重绘检查。

---

## 三、禁止事项

* **禁止文字套框**：关键词必须骑在线条上，不使用矩形、圆角胶囊或圆圈包裹。
* **禁止多词长句**：不能把分支写成短语、句子或段落。
* **禁止机械直线**：不能以直线、等宽线条作为标准有机分支。
* **禁止线字失配**：不能出现线过长、字溢出、图文与分支长度明显不匹配。
* **禁止拥挤堆叠**：不能为了容纳更多内容而牺牲留白和可读性。
* **禁止纯文字导图**：不能让整图缺乏图像、符号和视觉记忆点。

---

## 四、功能映射

| 原则 | 最低产品要求 |
| :--- | :--- |
| 横向纸张 | 默认 A3/A4 横向，带边界警示 |
| 中心图像 | 中心节点必须支持图像化中心；Phase 1 允许单色受控 SVG，长期严格模式优先生成多色图像 |
| 图像优先 | 提供图像、符号、手绘素材与 AI 生成能力 |
| 关键词大写 | 默认英文大写，可配置但严格模式下默认开启 |
| 一词一分支 | 输入限制、警示或 AI 拆分建议 |
| 曲线渐细 | 锥形曲线渲染引擎 |
| 线字等长 | 基于文字和图像尺寸的动态分支长度计算 |
| 多色编码 | 主分支颜色系统与继承规则 |
| 非线性关联 | 独立关联线工具 |
| 留白清晰 | 布局密度控制、碰撞检测、子图拆分提示 |
| 主图与子图 | 主图索引、下钻、回溯导航 |
| 云朵边界 | 分支包裹功能 |
| 空白线条 | 空白分支插入功能 |
| 复习与重绘 | 周期回顾与快速重绘检查流程 |

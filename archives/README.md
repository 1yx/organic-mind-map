# Archives

This directory stores old code, experiments, and documentation that should not be treated as the active Phase 2 product baseline.

## Contents

- `phase1/docs_old/`: historical Phase 1 product and technical documents.
- `phase1/editable-handdraw-canvas-demo/`: old standalone editable handdraw demo.
- `phase1/handdraw-branch-demo/`: old standalone handdraw branch demo.
- `openspec/specs_phase1/`: Phase 1 OpenSpec specs for CLI preview, OrganicTree, read-only SVG rendering, local preview, PNG export, and related MVP contracts.
- `openspec/changes_archive_phase1/`: archived Phase 1 OpenSpec changes.
- `openspec/prototype-editable-handdraw-canvas/`: old OpenSpec prototype change for the standalone handdraw canvas demo.
- `phase2-experiments/PHASE_2_1st_attempts/`: first Phase 2 SVG attempt, kept only as historical reference.
- `phase2-experiments/generated-svg-preview.png`: generated preview from the earlier SVG attempt.

## Active Sources

Use these as the current Phase 2 baseline instead:

- `docs/`
- `PHASE_2_2nd_attempts/`
- `PHASE_2_3rd_attampts/`

The remaining `openspec/` directory is intentionally left as a shell until Phase 2 specs are rewritten.
